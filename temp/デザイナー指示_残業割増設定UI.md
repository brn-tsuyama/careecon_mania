# デザイナー指示書: 残業割増設定 UI
作成: 2026-06-30
対象画面: 会社設定 > 労務費計算設定

---

## 設計前提

- ユーザーはデジタル弱者（System2: 熟考モード）
- 法的グレーゾーンがある（割増を実質出していない会社が存在する）
- **デフォルトはすべてOFF**（グレー運用の会社がいきなり計算されないように）
- 会社全体（一括）と社員個別の2階層で設定できる

---

## 画面構成

### 会社設定（一括）

```
┌──────────────────────────────────────────────────────────┐
│  会社設定 > 労務費計算                                     │
├──────────────────────────────────────────────────────────┤
│  割増賃金の自動計算                                        │
│  ⚠️ OFFの場合、残業時間は記録されますが金額に反映されません │
│                                                          │
│  ┌──────────────────────────────────────────────┐        │
│  │ 種類          │ 適用  │ 割増率  │ 対象条件    │        │
│  ├──────────────┼───────┼─────────┼────────────┤        │
│  │ 時間外割増    │ ○ OFF │ 25 %   │ 1日8h超     │        │
│  │ 深夜割増      │ ○ OFF │ 25 %   │ 22:00〜5:00 │        │
│  │ 休日割増      │ ○ OFF │ 35 %   │ 法定休日    │        │
│  │ 月60h超割増   │ ○ OFF │ 25 %   │ 月60h超分   │        │
│  └──────────────────────────────────────────────┘        │
│                                                          │
│  割増率は法定値をデフォルトとしています。                   │
│  会社の規定に合わせて変更できます。                         │
│                              [保存する]                   │
└──────────────────────────────────────────────────────────┘
```

**各行のインタラクション:**
- トグル(OFF→ON): 「この会社の全社員に適用されます。工事台帳の労務費が再計算されます。よろしいですか？」確認モーダル
- 割増率フィールド: 数値直接入力（適用ONのときのみ編集可能）
- 対象条件: テキスト表示のみ（編集不可・法定条件の説明）

---

### 社員個別設定（プロジェクトメンバー設定から）

**場所:** 案件 > メンバー > 各メンバーの詳細 > 労務費設定

```
┌──────────────────────────────────────────────────┐
│  田中 太郎 さんの労務費設定                        │
├──────────────────────────────────────────────────┤
│  単価設定                                         │
│  ¥6,500 / 時間（時間単価）                        │
│                                                  │
│  割増賃金                                         │
│  [✓ 会社設定に従う（現在: すべてOFF）]            │  ← デフォルト
│  [ ] 個別設定を使う                               │
│                                                  │
│  ↑ 「個別設定を使う」をONにすると以下が展開       │
│  ┌────────────────────────────────────┐          │
│  │ 種類       │ 適用  │ 割増率       │          │
│  │ 時間外割増 │ ○ OFF │ 25 %        │          │
│  │ 深夜割増   │ ○ OFF │ 25 %        │          │
│  │ 休日割増   │ ○ OFF │ 35 %        │          │
│  │ 月60h超    │ ○ OFF │ 25 %        │          │
│  └────────────────────────────────────┘          │
└──────────────────────────────────────────────────┘
```

**優先度ルール（システム内部）:**
```
個別設定あり → 個別設定を使う
個別設定なし → 会社設定を使う
```

---

## 割増の組み合わせ（エンジニア向け補足）

深夜 + 時間外が重なる場合は合算:
```
通常勤務: 1.0 × 時間単価
時間外のみ: 1.25 × 時間単価
深夜のみ: 1.25 × 時間単価
時間外 + 深夜: 1.5 × 時間単価  （重複分は加算）
休日: 1.35 × 時間単価
休日 + 深夜: 1.6 × 時間単価
```

---

## デザイン上の注意

1. トグルのON/OFFは色だけでなく文字でも示す（色覚対応 + System2対応）
2. 「変更すると再計算される」警告は保存ボタンを押したときにモーダルで出す（変更中は出さない）
3. 割増率の%入力は整数のみ許容（小数なし）
4. 設定画面は管理者のみアクセス可
